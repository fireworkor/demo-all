Based on Gemini Pro API.

考核要求：
1、安装一台虚拟机，操作系统为centos 8-stream，网卡选择nat模式，内存、cpu、磁盘大小可自行设置。安装后，关闭此虚拟机，以此虚拟机为模板克隆两台虚拟机hostA，hostB。（5）
2、给hostA主机添加一块大小为20G的硬盘，对此硬盘进行分区，使用GPT分区表，分区1大小为2G（类型为Linux swap），分区2大小为4G（类型为Linux LVM），分区3使用剩余所有空间（类型为Linux LVM）。（5分）
3、hostA主机使用分区1为系统增加交换分区的大小，并实现持久挂载。（5分）
4、hostA主机使用分区2创建卷组vg1，然后创建逻辑卷lv1，大小为2G。（5分）
5、hostA主机创建目录/mount_dir，将逻辑卷lv1格式化为xfs，挂载至/mount_dir目录，并配置持久挂载，并在/mount_dir目录中创建文件，验证目录可写。（10分）
6、hostA主机使用分区3的所有空间为系统根分区扩容。（10分）
7、将hostA、hostB的主机名分别配置为serverX和clientX，网段使用nat模式的网段（掩码24，如192.168.1.0/24），配置serverX地址最后一段为100，clientX地址最后一段为200（如192.168.1.100/24,192.168.1.200/24）；网关和dns根据实际情况自行配置，能通外网即可。（5分）
注：X为学号后两位
8、在两台主机上配置hostA、hostB的主机名解析。（5分）
9、配置hostA与hostB，使两台主机间可以互相免密登录，创建test1用户，两台主机仅允许普通用户test1在主机网卡所在的网段远程登录，且禁止root远程登录。（10分）
10、在hostB创建普通用户test2、test3，创建组grouptest，将test2用户和test3用户加入组grouptest，设置用户test2、test3最长密码使用有效期为365天，用户密码到期前，提前收到警告信息的天数为14天。（5分）
11、在hostB创建目录/sharedir，设置目录属主为root，属组为grouptest，设置目录权限为属主拥有所有权限，属组拥有读写权限，其他人具有只读权限。设置目录可以继承属组，且用户只能删除自己创建的文件，不能删除其他人的文件。（5分）
12、在hostB配置system journal，将记录永久保存。（5分）
13、配置hostA与hostB同步阿里云时钟同步服务器。（5分）
14、将hostB主机中/var/log目录下的所有内容打包，并使用gzip压缩，命名为varlog_bak.tar.gz，并将此tar包发送至hostA主机root用户的家目录。（5分）
15、使用centos 8-stream的光盘内容配置本地yum仓库，确保可以正常下载软件。（5分）
16、对各实验各关键部分进行必要说明及基本验证。（10分）

1. 安装虚拟机

  • 安装 VMware Workstation Pro 或 VirtualBox 等虚拟机软件。
  • 创建新虚拟机，操作系统选择 CentOS 8-Stream。
  • 配置网卡为 NAT 模式。
  • 设置内存、CPU 和磁盘大小（例如：内存 4GB、CPU 2 个内核、磁盘 60GB）。
  • 完成安装后，关闭虚拟机。

2. 克隆虚拟机

  • 右键单击原始虚拟机，选择“克隆”。
  • 创建两个克隆虚拟机，命名为 hostA 和 hostB。

3. hostA 添加硬盘

  • 右键单击 hostA 虚拟机，选择“添加硬盘”。
  • 选择“新建虚拟磁盘”，大小为 20GB。

4. hostA 磁盘分区

使用 parted 工具对新硬盘进行分区：

Copy 
parted /dev/sdb # 替换为实际的新硬盘设备名
mklabel gpt
mkpart primary linux-swap 1MiB 2048MiB
set 1 swap on
mkpart primary linux-lvm 2048MiB 6144MiB
mkpart primary linux-lvm 6144MiB 100%

5. hostA 增加交换空间

  • 创建交换分区：
Copy 
mkswap /dev/sdb1
swapon /dev/sdb1
  • 配置持久挂载：
Copy 
echo "/dev/sdb1 none swap defaults 0 0" >> /etc/fstab

6. hostA 创建逻辑卷

  • 创建卷组：
Copy 
vgcreate vg1 /dev/sdb2
  • 创建逻辑卷：
Copy 
lvcreate -n lv1 -L 2G vg1

7. hostA 挂载逻辑卷

  • 创建挂载目录：
Copy 
mkdir /mount_dir
  • 格式化逻辑卷：
Copy 
mkfs.xfs /dev/vg1/lv1
  • 挂载逻辑卷：
Copy 
mount /dev/vg1/lv1 /mount_dir
  • 配置持久挂载：
Copy 
echo "/dev/vg1/lv1   /mount_dir   xfs   defaults,nofail   0   0" >> /etc/fstab

8. hostA 扩容根分区

使用 LVM 扩容根分区：

  • 扩展卷组：
Copy 
vgextend vg1 /dev/sdb3
  • 扩展逻辑卷：
Copy 
lvextend -l +100%FREE /dev/vg1/lv0 # 逻辑卷 lv0 为根分区
  • 调整文件系统大小：
Copy 
xfs_growfs /dev/vg1/lv0

9. 配置主机名和网络

  • 设置主机名：
Copy 
hostnamectl set-hostname server<学号后两位>
  • 配置网络：
Copy 
nmcli con modify eth0 ipv4.addresses 192.168.1.100/24
nmcli con modify eth0 gateway 192.168.1.1
nmcli con modify eth0 dns 8.8.8.8

10. 配置主机名解析

  • 在 hostA 上创建 /etc/hosts 文件：
Copy 
echo "192.168.1.100 server<学号后两位>" >> /etc/hosts
echo "192.168.1.200 client<学号后两位>" >> /etc/hosts

11. 配置免密登录

  • 在 hostA 上创建用户 test1：
Copy 
useradd test1
passwd test1
  • 生成密钥对：
Copy 
ssh-keygen -t rsa
  • 复制公钥到 hostB 上：
Copy 
ssh-copy-id -o StrictHostKeyChecking=no test1@192.168.1.200
  • 限制远程登录用户：
Copy 
sed -i '/PermitRootLogin/s/^.*$/PermitRootLogin no/' /etc/ssh/sshd_config
sed -i '/AllowUsers/s/^.*$/AllowUsers test1/' /etc/ssh/sshd_config
systemctl restart sshd

12. hostB 创建用户和组

  • 创建用户 test2 和 test3：
Copy 
useradd test2
passwd test2
useradd test3
passwd test3
  • 创建组 grouptest：
Copy 
groupadd grouptest
  • 将用户添加到组：
Copy 
usermod -aG grouptest test2
usermod -aG grouptest test3
  • 设置密码过期策略：
Copy 
chage -M 365 test2
chage -W 14 test2
chage -M 365 test3
chage -W 14 test3

13. hostB 配置目录权限

  • 创建目录 /sharedir：
Copy 
mkdir /sharedir
  • 设置目录权限：
Copy 
chown root:grouptest /sharedir
chmod 750 /sharedir
setfacl -m u:test2:rwx /sharedir
setfacl -m g:grouptest:rw /sharedir
setfacl -m o::r /sharedir
  • 设置目录继承属组和文件删除限制：
Copy 
setfacl -d -m g:grouptest:rwx /sharedir
setfacl -d -m u:test2:rwx /sharedir/

14. hostB 配置 systemd journal

  • 配置 systemd journal 永久保存：
Copy 
sed -i '/Storage/s/^.*$/Storage=persistent/' /etc/systemd/journald.conf
systemctl restart systemd-journald

15. hostA 与 hostB 同步阿里云时间服务器

  • 安装 chrony：
Copy 
yum install chrony
  • 配置 chrony：
Copy 
sed -i 's/^.*pool.*/pool ntp.aliyun.com iburst maxsources 4/' /etc/chrony.conf
systemctl restart chronyd

16. hostB 备份并发送至 hostA

  • 备份 /var/log 目录：
Copy 
tar -czvf varlog_bak.tar.gz /var/log
  • 发送备份文件至 hostA：
Copy 
scp varlog_bak.tar.gz root@192.168.1.100:/root

17. 配置本地 yum 仓库

  • 挂载光盘镜像：
Copy 
mount -o loop /path/to/CentOS-8-Stream-x86_64-dvd1.iso /mnt
  • 创建本地仓库：
Copy 
yum install createrepo
createrepo /mnt
  • 配置 yum 源：
Copy 
vi /etc/yum.repos.d/localrepo.repo
[localrepo]
name=Local CentOS-8-Stream Repository
baseurl=file:///mnt
enabled=1
gpgcheck=0
Regenerate
